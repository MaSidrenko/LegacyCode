# LegacyCode
------------------------

### Имя проекта:
"Legacy Code Simulator" (Временно)

------------------------

### Основная идея проекта
Веб-приложение, где пользователю дают кусок плохо написанного, запутанного кода (Legacy кода), с некоторыми багами
на каком либо языке (Изначально планируеться поддержка C# и Python).
Пользователю нужно:  
  1. Найти и исправить баги;  
  2. Добавить нову фичу в эту запутанную структуру;  
  3. Написать тесты для этого кода;  

-----------------------

### Фишка
Развитие критически важного навыка, в виде рефакторинга кода  

-----------------------

### Стек используемых технологий
#### Бекенд:
HTTP сервера:  
Python (Fast API, SQLAlchemy) - Оценка кода(Возможна работа с ИИ), работа с БД.   
C# (ASP.NET Web API,unit test) - Принимает request от фронтенда, запускает код, делает тесты и отправляет response питону на оценку
#### Фронтенд:
React.js, HTML, CSS
#### База Данных:  
SQLite или PostgreSQL

--------------------------
